EF - entity framework
Фреймворк для упрощения работы с бд. Главная идея - перенос работы с БД на яп (прикрытия SQL), 
унификация - создание одинаковых способов для работыс разными субд
Базируется на ADO, не имеет заменных способов доступа к данными.
"+" универсальность, один яп (не нужно учить SQL)
"-" сложность, отсутствие "фич" SQL 

Установка
Необхожимо добавить несколько NUGET пакетов
- Microsoft.EntityFrameworkCore - ядро фреймворка, основные инструменты
- Microsoft.EntityFrameworkCore.SqlServer - драйвер для MS SQL Server
- Microsoft.EntityFrameworkCore.Tools - командна консоль
